Deploy the backend

The Forge script is pre-loaded in your copied sheet. Open Apps Script and deploy it as a web app. Takes about 2 minutes with the step-by-step guide.

1 In your copied sheet, click Extensions → Apps Script — the Forge code is already loaded
2 Click Deploy → New deployment
3 Set type to Web app. Set Execute as: Me. Set Who has access: Anyone
4 Click Deploy. Approve the Google permissions popup
5 Copy the Web app URL — it starts with https://script.google.com/macros/s/ — you will need this in Step 3
